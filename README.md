# ⚽ Predictor Mundial 2026

Una aplicación web elegante y funcional para predecir resultados del Mundial de Fútbol 2026. Diseñada con atención a los detalles de UX/UI y máxima funcionalidad.

## 🎯 Características

- ✅ **Predicciones Completas**: Grupos, Octavos, Cuartos, Semifinales y Final
- ✅ **Datos Correctos**: Todos los 8 grupos y 64 partidos del Mundial 2026
- ✅ **Exportación a Excel**: Descarga tus predicciones con toda la información del usuario
- ✅ **Gestión de Usuario**: Captura nombre, email, equipo favorito y teléfono
- ✅ **Almacenamiento Local**: Guarda automáticamente en localStorage
- ✅ **Diseño Elegante**: Interfaz moderna con gradientes y animaciones suaves
- ✅ **Estadísticas en Tiempo Real**: Seguimiento de predicciones completadas
- ✅ **Información de Estadios**: Detalles de todos los estadios del torneo
- ✅ **Completamente Responsivo**: Funciona perfectamente en móvil, tablet y desktop

## 📱 Compatibilidad

- Chrome/Edge (última versión)
- Firefox (última versión)
- Safari (última versión)
- Navegadores móviles modernos

## 🚀 Cómo Usar

1. **Descargar o clonar** el repositorio
2. **Abrir** `index.html` en tu navegador
3. **Completar datos** del predictor (nombre, email, etc.)
4. **Hacer predicciones** en cada partido
5. **Exportar a Excel** cuando lo desees

## 📊 Exportación de Datos

Al hacer clic en "Exportar", se genera un archivo Excel con:
- Hoja 1: Información del predictor (nombre, email, equipo favorito, teléfono)
- Hoja 2: Todas tus predicciones con detalles del partido

## 🎮 Fases del Torneo

- **Grupos (A-H)**: 48 partidos
- **Octavos**: 8 partidos
- **Cuartos**: 4 partidos
- **Semifinales**: 2 partidos
- **Final**: 2 partidos (Final + Tercer Puesto)

**Total: 64 partidos**

## 🚀 Deployment

### Opción 1: GitHub Pages
1. Fork o crea un repositorio con el nombre `username.github.io`
2. Sube `index.html` a la raíz
3. Accede a `https://username.github.io`

### Opción 2: Cualquier servidor web
1. Sube `index.html` a tu servidor
2. Accede via HTTP/HTTPS

### Opción 3: Vercel/Netlify
1. Conecta tu repositorio de GitHub
2. Deploy automático con HTTPS
