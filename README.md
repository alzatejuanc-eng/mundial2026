# 🏆 Mundial 2026 - Predictor de Resultados

**Versión: Raíz + Vercel (Estructura Simplificada)**

> Una aplicación web para predecir resultados de los 88 partidos del mundial, con sistema de puntos automático y explorador de estadios.

## 🚀 Despliegue Rápido

### ✅ Características

- ✅ **88 partidos** del mundial en una sola carpeta
- ✅ **Sin subcarpetas** (todo en raíz)
- ✅ **Deploy en Vercel** (automático desde GitHub)
- ✅ **48 equipos** y **16 estadios** incluidos
- ✅ **Sistema de puntos** automático
- ✅ **Almacenamiento local** en navegador
- ✅ **Diseño premium** con colores ExO
- ✅ **Responsive** en todos los dispositivos

## 📋 Estructura (Todo en Raíz)

```
./
├── index.html          ⭐ Página principal

├── styles.css          ⭐ Diseño completo
```

**Todo en una carpeta. Sin subcarpetas innecesarias.**

## 🎯 Deploy en Vercel (2 Pasos)

### Paso 1: GitHub
1. Crea repositorio en GitHub
2. Sube los archivos (TODO en raíz)
3. Push a main branch

### Paso 2: Vercel
1. Ve a vercel.com
2. Import → Selecciona repositorio
3. Deploy ✅

**Tu URL:** `https://mundial2026-XXXXX.vercel.app`
